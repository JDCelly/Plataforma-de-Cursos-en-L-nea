Nivel 4 — DTOs y Validación

Los DTOs protegen la API validando cada petición antes de que llegue al Controller.

12
¿Qué combinación de ValidationPipe está bien?
Configuración de main.ts
En main.ts, la app tiene new ValidationPipe({ whitelist: true, forbidNonWhitelisted: true, transform: true }). Si un cliente envía {"nombre":"X", "hackerField": "malicioso"}, ¿qué sucede?
💡 Pista: whitelist: true elimina campos extras. forbidNonWhitelisted: true va más allá: lanza un error HTTP 400 si hay campos no definidos en el DTO.
